# Vitamin

VI TextAdept Mode INterface. Your daily Textadept editing supplement.

Implements vi keybindings as a mode in Textadept.

vi/ex motion and command bindings are somewhat esoteric and non-intuitive, but for users that are already used to them, they can be a highly efficient way of navigating and modifying a document.

## Implemented

h j k l w W b B e E H M L G 0 $ ^ + - | % _ ( ) { } ctrl+f ctrl+b ctrl+d ctrl+u ctrl+e crtl+y ctrl+h backspace enter ctrl+j ctrl+n ctrl+p space

## Not Implemented

Any commands not listed above, including
- ex commands
- search
- input mode
